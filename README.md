# Свои мастера — Йошкар‑Ола (backend)

Локальный сервис для жителей Йошкар‑Олы, который помогает найти мастеров по ремонту, уборке и красоте на дому.  
Этот репозиторий содержит бэкенд (REST API) на Spring Boot.

## Цель MVP

Сделать минимально рабочую версию, которая позволяет:

- регистрировать мастеров и заказчиков;
- создавать заявки в разделах «Ремонт», «Уборка», «Красота»;
- получать отклики мастеров и выбирать исполнителя;
- оформлять сделку и принимать онлайн‑оплату с комиссией сервиса 5 %;
- отмечать выполнение заказа и оставлять отзывы.
